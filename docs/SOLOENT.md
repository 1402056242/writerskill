# SOLOENT.md — 项目控制面板

> **⚠️ 流程驱动声明**  
> 本项目使用 **武行长篇小说技能包** 驱动创作流程。  
> 
> **强制约束：**
> 1. 任何 AI Agent 在读取本文件时，必须**立即读取总控文件**：  
>    `.soloent/skills/wuhang-long-all/SKILL.md`  
>    以了解完整的「边界 → 设定 → 大纲 → 写作 → 审查 → 体检」流程。
> 2. 后续所有创作步骤必须遵循技能包内定义的标准流程，各阶段技能文件位于：  
>    `.soloent/skills/wuhang-long-all/`：
>    - 边界阶段：`wuhang-boundary.md`（样板书拆解）
>    - 设定阶段：`wuhang-ideation.md`（创意与设定）
>    - 大纲阶段：`wuhang-outline.md`（总纲 → 卷纲 → 章纲）

> 3. **严禁脱离技能包流程自创步骤**。  
> 4. 不确定下一步时，请优先参考本文件 §8（项目进度与待办），或主动询问用户调用哪个子技能。

- **角色状态文件** → `.novelkit/memory/character_state.md`

## 9. 使用提示

> 当你或 Agent 不确定「现在应该干什么」时：

1. 先看 §8.1 的「当前阶段 / 核心任务 / 推荐 Skill」；
2. 再打开 `.soloent/skills/wuhang-long-all/SKILL.md` 对应子 skill 的说明；
3. 然后按 skill 文档里的步骤逐条执行。

> 当你怀疑「项目可能已经有点乱」时：

- 运行 `project_doctor.py`（项目体检脚本），让它检查：
  - 目录结构是否完整；
  - SOLOENT 中的指针是否都能找到实际文件；
  - commits 与 正文 / 大纲 / memory 是否有明显断档。
