## 禁用词表（出现即标记）

### 情绪/状态直告

- 眼中闪过、眼底闪过、眼神闪过
- 不易察觉的、微不可察、一丝情绪
- 深吸一口气、长舒一口气
- 嘴角勾起、嘴角微扬、嘴角上扬
- 浮现出一抹、嘴角一抹/一丝弧度/冷笑/玩味
- 脸上带着笑意、脸上堆满了笑、显得有些兴奋
- 眼神热切、目光里毫不遮掩、XXX的眼神
- 心中一动、心头一震、心中涌起、心中一凛、心中暗道
- 捏紧了拳头、指节因用力而发白
- 死死盯着、瞳孔剧烈收缩、瞳孔收缩
- 不禁、不由得、不由自主
- 涌上心头、涌现出来
- 眼眸深邃、眼神复杂、目光复杂

### 书面腔/文言腔/形容词

- 仿佛、犹如、宛若、如同（单用作比喻连词时）
- 缓缓（作动作状语时）、徐徐
- 不容置疑、显而易见、毋庸置疑
- 锐利的、纯粹、彻骨的寒
- 淬毒、冰冷、深邃、森然、轮廓分明的下颌、难以置信的、死寂
- 恐怖如斯
- 岁月流逝、光阴似箭

### AI高频套路词

- 沉默片刻、沉默良久
- 轻声开口、缓缓开口
- 微微一愣（高频时）
- 莫名其妙地感到
- 说道、问道、笑道（高频时，尽量用动作替代）
- 瞬间（用作情绪/反应状语时）
- 涟漪、轰然、死亡的阴影

### 解释腔/上帝感

- 之所以……是因为
- 她/他不知道的是
- 殊不知
- 多年以后（用于回忆预示时）
- 仿佛预示着
- 这意味着
- 原来……（用于作者解释时）
- 他相信 / 他知道
- 不得不说 / 毋庸置疑
- 电光火石之间 / 说时迟那时快
- 演得真好、他就是这样薄情（替读者定性）

### 陈词滥调/烂比喻

- 倒吸一口凉气（除非真的冷）
- 剑眉星目 / 面如冠玉
- 深邃的眼眸 / 深邃的眸子 / 淬毒的眸子 / 冰冷的声音
- 空气仿佛凝固了、黑夜如墨 / 夜色如墨
- 心中翻滚 / 咆哮 / 绝望
- 死死地钉在 XX

## 自然文本基准（网文对比参考）

| 维度 | 自然网文 | AI味文本 |
|------|---------|----------|
| 对话标签 | 60%+无标签，用动作替代「说」 | 几乎每句都有「说道/问道」 |
| 情绪表达 | 动作展示（「手在抖」） | 直接告诉（「很紧张」） |
| 比喻 | 生活化（「像哈士奇护食」） | 文学化（「如寒冰般」） |
| 排比 | 偶尔1-2个，从不连续3+ | 连续3-5个是标配 |
| 结尾 | 动作/对话收尾 | 总结/升华/感慨收尾 |
| 段落长度 | 1-3句为主，偶尔1句独占一行 | 每段4-6句，整齐均匀 |
| 语气词 | 「嗯」「哦」「行吧」「靠」 | 几乎没有 |
| 省略 | 大量省略，读者自己脑补 | 面面俱到，生怕读者不懂 |

## 自然替换速查

| 原词/句 | 替代方案 |
|--------|----------|
| 深吸一口气 | 胸口起伏了一下 / 直接删 |
| 眼中闪过一丝…… | 他垂下眼 / 眯起眼 |
| 嘴角勾起一抹冷笑 | 笑了一下，没到眼底 / 乐了 |
| 仿佛…… | 像… / 直接白描 |
| 不禁…… | 直接写动作 |
| 缓缓开口 | 「说」/ 用动作引出对话 |
| 他很[情绪词] | 写对应的身体反应/动作 |
| 她不知道的是 | 直接删 |
| 之所以…是因为 | 直接删，因果让读者从动作对话里自己拼 |
| 这一刻他知道…… | 直接删，改动作/场景收尾 |
| 连续排比3段以上 | 保留最强1-2个，其余删或改 |
| 瞳孔收缩 | 他停了一下 / 直接跳到反应动作 |
| 恐怖如斯 | 直接删，用受害方或环境的具体反应替代 |
| 瞬间（作情绪状语） | 直接删，或改为具体时间动作 |

## 白名单说明

以下情况**豁免**禁用词标记：

- 角色口头禅 / 绰号中恰好包含禁用词（如角色名叫「缓缓」）
- 世界观专有名词（如功法名「仿佛诀」）
- 作者刻意使用的修辞手法，且上下文明确

遇到豁免情况，在去AI味报告中注明，不计入命中次数。

## 结构级 AI 病症（段落/场景层）

> 词汇级禁用词管「用了什么词」，本节管「写了什么结构」。两者都命中才算真正去干净。

### 病症一：剧情梗概冒充正文

**表现**：一段话连续交代发生了什么，却没有现场动作。角色决定、行动、成功都在同一段里完成。读者知道结果，但没有经历压力。

**修法**：把「结果句」拆成触发 → 尝试 → 阻碍 → 再尝试 → 代价。让人物在现场处理一个具体问题，而不是让作者替他总结。

> ❌「林舟很快意识到有人在陷害他，于是决定反击。他冷静地找到证据，成功揭穿了对方。」  
> ✅ 把过程拆开：他翻那张采购单，指腹停在纸角，墨迹还没干，签名却是三天前的日期——现在退一步，罪名就会落死；往前一步，藏在后面的人也许会先动手。

### 病症二：冲突没有代价

**表现**：主角遇到麻烦，马上想到办法，马上成功。失败只是装饰，不改变资源、关系或心态。

**修法**：至少给一次错误判断或不完整胜利。让胜利带来新问题——暴露身份、欠下人情、失去证据、激怒对手。爽点来自「受压 → 反击 → 付代价 → 赢一点」，不是来自作者直接宣布人物很强。

### 病症三：反派/配角工具化

**表现**：反派只会嘲讽，配角只会夸主角。所有人都围绕主角展示爽点，没有自己的利益和算盘。

**修法**：给反派一个合理目标（哪怕目标很坏），给配角一个小算盘、小恐惧或小善意。人物不能只是功能按钮。

### 病症四：章末机械预告

**表现**：「他不知道的是，真正的危机才刚刚开始。」「一场更大的风暴即将到来。」「命运的齿轮开始转动。」

**修法**：用具体信息改变局势收尾——一个动作、一件物品、一句话、一个人的出现。章末钩子要改变读者对当前局势的判断，而不是空泛宣布「还有更多」。

> ✅ 示例：「门外忽然响了一声轻咳。林舟抬头，看见母亲站在廊下，手里攥着的，正是他昨夜烧掉的那半封信。」

### 病症五：爽点直接宣布

**表现**：「所有人都震惊了。」「他终于逆袭了。」「这是属于他的高光时刻。」

**修法**：不宣布爽，让读者从对手反应、现场静默、利益翻转里自己感到爽。作者宣布爽点等于替读者跳过了爽的过程。

### 病症六：关系变化没有余波

**表现**：吵架后下一段恢复正常。救命、背叛、告白、误会没有改变后续说话方式和相处距离。

**修法**：在下一场保留一点残留——避开目光、称呼变化、刻意客气、突然沉默。小事也要留下关系变化、心理残留或新的误会种子。
