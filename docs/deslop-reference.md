# 去AI味参考：禁用词表 & 自然文本基准

> 供 `wuhang-write.md` 步骤 3.5「去AI味收尾」按需加载。

---

## 禁用词表（出现即标记）

### 情绪/状态直告

- 眼中闪过、眼底闪过、眼神闪过
- 不易察觉的、一丝情绪
- 深吸一口气、长舒一口气
- 嘴角勾起、嘴角微扬、嘴角上扬
- 心中一动、心头一震、心中涌起
- 不禁、不由得、不由自主
- 涌上心头、涌现出来
- 眼眸深邃、眼神复杂、目光复杂

### 书面腔/文言腔

- 仿佛、犹如、宛若、如同（单用作比喻连词时）
- 缓缓（作动作状语时）、徐徐
- 不容置疑、显而易见、毋庸置疑
- 岁月流逝、光阴似箭

### AI高频套路词

- 沉默片刻、沉默良久
- 轻声开口、缓缓开口
- 微微一愣（高频时）
- 莫名其妙地感到
- 说道、问道、笑道（高频时，尽量用动作替代）

### 解释腔/上帝感

- 之所以……是因为
- 她/他不知道的是
- 殊不知
- 多年以后（用于回忆预示时）
- 仿佛预示着
- 这意味着
- 原来……（用于作者解释时）
- 演得真好、他就是这样薄情（替读者定性）

## 自然文本基准（网文对比参考）

| 维度 | 自然网文 | AI味文本 |
|------|---------|----------|
| 对话标签 | 60%+无标签，用动作替代「说」 | 几乎每句都有「说道/问道」 |
| 情绪表达 | 动作展示（「手在抖」） | 直接告诉（「很紧张」） |
| 比喻 | 生活化（「像哈士奇护食」） | 文学化（「如寒冰般」） |
| 排比 | 偶尔1-2个，从不连续3+ | 连续3-5个是标配 |
| 结尾 | 动作/对话收尾 | 总结/升华/感慨收尾 |
| 段落长度 | 1-3句为主，偶尔1句独占一行 | 每段4-6句，整齐均匀 |
| 语气词 | 「嗯」「哦」「行吧」「靠」 | 几乎没有 |
| 省略 | 大量省略，读者自己脑补 | 面面俱到，生怕读者不懂 |

## 自然替换速查

| 原词/句 | 替代方案 |
|--------|----------|
| 深吸一口气 | 胸口起伏了一下 / 直接删 |
| 眼中闪过一丝…… | 他垂下眼 / 眯起眼 |
| 嘴角勾起一抹冷笑 | 笑了一下，没到眼底 / 乐了 |
| 仿佛…… | 像… / 直接白描 |
| 不禁…… | 直接写动作 |
| 缓缓开口 | 「说」/ 用动作引出对话 |
| 他很[情绪词] | 写对应的身体反应/动作 |
| 她不知道的是 | 直接删 |
| 之所以…是因为 | 直接删，因果让读者从动作对话里自己拼 |
| 这一刻他知道…… | 直接删，改动作/场景收尾 |
| 连续排比3段以上 | 保留最强1-2个，其余删或改 |

## 白名单说明

以下情况**豁免**禁用词标记：

- 角色口头禅 / 绰号中恰好包含禁用词（如角色名叫「缓缓」）
- 世界观专有名词（如功法名「仿佛诀」）
- 作者刻意使用的修辞手法，且上下文明确

遇到豁免情况，在去AI味报告中注明，不计入命中次数。
